Consulta de Inventário a ligar ao documento...
O widget vai ler automaticamente todas as tabelas do documento, para poder cruzar dados entre elas (ex: Inventário + Entradas + Saídas + Fornecedores).
Chave API do Gemini (grátis, necessária para as respostas)
Obtida grátis em aistudio.google.com/apikey (sem cartão de crédito, com limite de pedidos por minuto). Fica só em memória enquanto este widget está aberto — não é guardada em disco. Nota: no nível gratuito, a Google pode usar os teus pedidos para melhorar os seus produtos.
